# 📝 React To-Do App (with TypeScript)

A beginner-friendly to-do list app built with **React** and **TypeScript**.

## 🚀 Features

- Add tasks
- View list of tasks
- Mark tasks as completed (coming soon)
- Delete tasks (coming soon)

## 🛠️ Tech Stack

- React
- TypeScript
- JavaScript (JSX/TSX)
- CSS (or add Tailwind later if you want)
